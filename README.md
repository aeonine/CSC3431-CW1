# Evolutionary Optimisation Algorithms

Comparing evolutionary and swarm-based optimisation approaches through parameter tuning, convergence analysis and benchmark optimisation experiments.

University of Newcastle CSC3431 Biological Computation coursework.

[📄 View report](./report) [💻 View source](./code)

---

## Overview

This project explores biologically inspired optimisation algorithms through the implementation and evaluation of:

- Genetic Algorithms (GA)
- Particle Swarm Optimisation (PSO)

The algorithms are compared using benchmark optimisation problems to investigate convergence behaviour, parameter sensitivity and optimisation performance.

---

## Results

The experiments analyse how different optimisation strategies perform under varying parameters, highlighting differences in exploration, convergence speed and solution quality.
